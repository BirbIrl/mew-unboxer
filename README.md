# Mewgenics unpacker for mewbox


required dependencies: [ffdec](https://github.com/jindrapetrik/jpexs-decompiler)

1. unpack the game using [GPAK-Extractor](https://github.com/ShootMe/GPAK-Extractor) (Works just fine through wine if you're on linux)
2. run the lua script

the output is made to be read by the mewbox data web viewer
